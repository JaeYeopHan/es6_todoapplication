# Todo Application with ECMAScript6

## Intall
```bash
# If your local does not have a yarn installed
$ brew install yarn

$ mkdir todo
$ cd todo
$ git clone https://github.com/JaeYeopHan/es6_todoapplication
$ cd es6_todoapplication
$ yarn install
```
---

## Test
```bash
$ npm test
```
## Run
```bash
npm run start
```
for dev
```bash
npm run dev
```
